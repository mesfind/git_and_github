Basic Configuration

Task Command
Set user name git config --global user.name "Your Name"
Set email git config --global user.email "your.email@example.com"
Set default branch git config --global init.defaultBranch main
Set text editor git config --global core.editor "nano -w"
View all settings git config --list --global
Get help git help or git <command> -h

Repository Management

Task Command
Create new repository git init
Clone a repository git clone <url>
Clone with shallow history git clone --depth 1 <url>
Check repository status git status
View ignored files git status --ignored

Tracking Changes

Task Command
Add file to staging git add <file>
Add all files git add .
Add directory git add <directory>/
Commit changes git commit -m "message"
Commit all tracked changes git commit -a -m "message"
View commit history git log
Compact history git log --oneline
History with graph git log --oneline --graph
Limit commits git log -n

Comparing and Viewing Changes

Task Command
View unstaged changes git diff
View staged changes git diff --staged
View changes between commits git diff <commit1> <commit2>
View specific file changes git diff <commit> -- <file>
Show commit details git show <commit>

Undoing Changes

Task Command
Discard unstaged changes git restore <file>
Discard all unstaged changes git restore .
Unstage a file git restore --staged <file>
Restore file from commit git restore --source <commit> <file>
Undo last commit (keep changes) git reset --soft HEAD~1
Undo last commit (unstage changes) git reset HEAD~1
Undo last commit (discard everything) git reset --hard HEAD~1
Revert a commit (new commit) git revert <commit>

Task Command

View remotes git remote -v
Rename remote git remote rename <old> <new>
Remove remote git remote remove <name>
Push to remote git push origin main
Pull from remote git pull origin main
Fetch without merging git fetch origin main
Set upstream git push -u origin main

!important.csv Exception to previous rule
*.csv All CSV files
**/*.nc NetCDF files anywhere

__pycache__/ Python cache
venv/ Virtual environment

Common Workflows

Basic Collaborative Workflow

git pull origin main
# make changes
git add <files>
git commit -m "description"
git push origin main

Branching for Experiments

git checkout -b experiment/hyperopt
# make experimental changes
git add .
git commit -m "exp: hyperparameter tuning"
git push -u origin experiment/hyperopt
# later merge to main
git checkout main
git merge experiment/hyperopt

Resolving Conflicts

git pull origin main
# edit conflicting files
git add <resolved-files>
git commit -m "merge: resolve conflicts"
git push origin main

Cleaning Notebook Outputs

jupyter nbconvert --ClearOutputPreprocessor.enabled=True --inplace notebook.ipynb
git add notebook.ipynb
git commit -m "clean notebook outputs"

Git Aliases for Data Scientists

Add to ~/.gitconfig:

[alias]
    graph = log --graph --oneline --all
    wip = commit -am "WIP"
    unstage = reset HEAD --
    nb-clean = !jupyter nbconvert --ClearOutputPreprocessor.enabled=True --inplace
    sync = !git pull --rebase && git push
    lfs-ls = lfs ls-files
    last = log -1 HEAD
    undo = reset --soft HEAD^

Troubleshooting

Problem Solution
Push rejected git pull origin main then push again
Merge conflicts Edit files, git add, git commit
Forgot to stage git commit -a -m "message"
Wrong commit message git commit --amend -m "new message"
Can’t push to GitHub Check SSH: ssh -T git@github.com
Proxy issues git config --global --unset http.proxy
